Description

The current activity log component is experiencing severe performance degradation when rendering projects with over 500 historical events. Scrolling becomes choppy, and memory usage spikes on lower-end devices.

Requirements:

  • Implement @tanstack/react-virtual for the main timeline container.
  • Ensure dynamic row heights are supported (comments vs simple status changes).
  • Maintain the current "sticky" date headers as the user scrolls.
  • Add a skeleton loading state for items fetched via infinite scroll.
